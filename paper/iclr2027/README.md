# ICLR 2027 Paper Workspace

This is an answer-first draft for ICLR 2027. The draft is written in a complete paper style while experiments are still pending.

Important:

- Assumed numerical results live in `assumed_results.json`.
- Generated tables and figures live under `tables/` and `figures/`.
- Regenerate them with `python3 ../../scripts/make_paper_assets.py` from the repository root.
- Replace assumed values with actual experiment outputs before submission.

Build:

```bash
python3 ../../scripts/make_paper_assets.py
tectonic main.tex
```
