# CVPR 2027 Secondary Positioning

The CVPR version should keep the same benchmark and method but shift emphasis:

- Make the benchmark construction and visual evidence stronger.
- Expand video-language and anomaly-detection comparisons.
- Add more qualitative multi-stream timelines and failure cases.
- Emphasize dataset curation, reproducible protocols, and visual cost-performance frontiers.
- Move agentic-decision formalism into a compact method section.

The current ICLR draft is the source of truth. This directory is a staging area for a later CVPR-formatted fork once the official CVPR 2027 style and dates are available.
